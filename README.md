# 🚀 Vest Soft

Консольный менеджер для Telegram: управление аккаунтами, создание групп/каналов/ботов, масс-рассылки и авто-реакции.

## 📱 Возможности

*   **Аккаунты:** добавление до 10 аккаунтов, авторизация по номеру, поддержка 2FA.
*   **Группы и каналы:** массовое создание (до 50), шаблоны названий, автоархивация.
*   **Боты:** создание через @BotFather (до 5), ручной или автоматический ввод имени.
*   **Рассылка:** до 5 аккаунтов и 20 чатов, последовательный или рандомный режимы, настраиваемые задержки.
*   **Авто-реакции:** ❤️ на новые сообщения, до 5 аккаунтов и 10 чатов, работа в фоне.

## ⚙️ Установка

**Обычный терминал (Linux/macOS/Windows):**
```bash
git clone https://github.com/vanyayt777777-source/Vest-Soft.git
cd Vest-Soft
pip install pyrogram
python vest.py
```

Termux (Android):

```bash
pkg update && pkg upgrade -y
pkg install python git -y
git clone https://github.com/vanyayt777777-source/Vest-Soft.git
cd Vest-Soft
pip install pyrogram
python vest.py
```

🎯 Первый запуск

При первом запуске потребуются API ID и API Hash. Получить их можно на my.telegram.org.
